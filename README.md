### Added

- Initial release with 250+ MCP tools
- **Google Ads** — 150+ tools for campaign management, keyword research, bidding, extensions, and analytics
- **Meta Ads** — 80+ tools for campaign CRUD, audiences, creatives, lead gen, and catalogs
- **Google Analytics 4** — 20+ tools for reporting, audiences, property config, and integrations
- **Research Tools** — Google search, webpage analysis, stock images, landing page analysis
- Setup guides for ChatGPT, Claude, Cursor, Windsurf, n8n, Codex CLI, and Gemini CLI
- Ready-to-import config files for all supported AI assistants
- n8n workflow template for automation
- Cross-platform analysis capabilities
